ABOUT THIS LEARNING PATH

This series of courses will teach you how to create a mobile healthcare application that retrieves and displays blood pressure information using HealthShare’s interoperable APIs.

LEARNING OBJECTIVES

After taking this course, you will be able to:

  • Retrieve patient data from an electronic medical record.
  • Create FHIR resources from this data.
  • Make the data available via RESTful services.
  • Create a simple mobile app to display the patient’s results.
  • Push the app to mobile devices.

FAQs

How long will the entire learning path take to complete?

The individual courses in the learning path range from 30 minutes to 2 hours. If you took the courses back to back, the entire learning track would take approximately 6 hours to complete.

What software/hardware is required to take an online course?

  • The courses work best with current versions of Chrome, Firefox or Safari, or with Internet Explorer version 10 or above.
  • For the majority of the courses, there is no software that is required as we have built in simulations that will step you through what is needed to push to both an iOS and Android device.
  • Adobe Flash player 10 to complete the simulations.

What do I need push the app to iOS and/or Android devices?

  • If you would like to push the application to an iOS device, you will need at least a Mac and an Apple Developer license.
  • If you would like to push an application to an Android device, steps will be provided in this course to install the necessary software to do this.

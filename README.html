Theme purpose

This is a temporary edX Dogwood theme made by Appsembler for NYIF.

Theme Authoring

To customize your theme: - Fork this repository. - Clone it into the theme directory next to your edx-platform directory and rename the theme directory to your new theme’s name. - Edit the .scss file in static/sass/ and rename the file with your theme’s name. - Edit the lms.envs.json file in edx-platform and set ‘USE_CUSTOM_THEME’ to true, and ‘THEME_NAME’ to your theme’s name. - Play around with the theme-variables.html template and basic-branding.scss files :)
